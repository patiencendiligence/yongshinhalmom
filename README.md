# Yongshin Halmom – AI Lifestyle Report

Yongshin Halmom is a web-based application that generates personalized lifestyle reports using AI.

## Overview

This product provides structured lifestyle insights based on user-provided information such as date and time input.  
The goal is to help users better understand their behavioral patterns and reflect on their daily habits.

The report is generated instantly and delivered as a digital product.

## Features

- AI-generated personalized lifestyle report
- Instant report generation after user input
- Simple and accessible web interface
- Preview (limited report) available before purchase
- Full report unlocked after payment

## How It Works

1. User enters basic input data  
2. System processes the input using an AI prompt  
3. A structured lifestyle report is generated  
4. Users can view a limited preview  
5. Full report is available after completing payment  

## Pricing

- One-time purchase (no subscription)
- Users pay once to unlock the full report


## Disclaimer

This product provides lifestyle insights for informational and self-reflection purposes only.  
It does not offer predictions, guarantees, or professional advice.

## Tech Stack

- Frontend: React (Vite)
- Backend: Supabase
- Hosting: Vercel
